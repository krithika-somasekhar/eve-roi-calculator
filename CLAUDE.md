# EVE Health ROI Calculator — Project CLAUDE.md

## Project Overview
This is the ROI Calculator for EVE Health's sales team. Prospects input their current metrics and get an estimated ROI from implementing the EVE Platform.

## Purpose
Lead magnet for the sales funnel. Prospects share contact info to get personalized ROI results.

## Key Inputs
- Current number of support tickets/month
- Average resolution time (hours)
- Number of FTE analysts
- Current tool cost (monthly)
- Number of facilities

## Key Outputs
- Estimated time savings (hours/month)
- Estimated cost savings ($/month)
- ROI percentage
- Payback period (months)

## Tech Stack
- Next.js or vanilla HTML/JS (student's choice)
- Tailwind CSS for styling
- Chart.js for ROI projections
- No backend — all calculations client-side

## Design Guidelines
- EVE Health brand colors: #1e40af (primary blue), #059669 (green for positive metrics), #dc2626 (red for pain points)
- Clean, professional, healthcare-appropriate design
- Mobile-responsive
- Accessibility: WCAG 2.1 AA compliant

## Deployment
- Vercel (free tier)
- Custom domain optional
